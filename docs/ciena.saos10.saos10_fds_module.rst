***********************
ciena.saos10.saos10_fds
***********************

**Manage forwarding domains on Ciena SAOS 10 devices**


Version added: 1.5.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module provides declarative management of a forwarding domain



Requirements
------------
The below requirements are needed on the host that executes this module.

- ncclient (>=v0.6.4)
- xmltodict (>=0.12.0)

Examples
--------

.. code-block:: yaml

    # Using merged

    - name: Configure forwarding domain
      ciena.saos10.saos10_fds:
        config:
          - name: remote-fd
            mode: vpls
            initiate-l2-transform:
              vlan-stack:
                - tag: 1
                  push-tpid: tpid-8100
                  push-pcp: map
                  push-vid: 127
        state: merged


    # Using overridden

    - name: Configure forwarding domain
      ciena.saos10.saos10_fds:
        config:
          - name: remote-fd
            mode: vpls
            initiate-l2-transform:
              vlan-stack:
                - tag: 1
                  push-tpid: tpid-8100
                  push-pcp: map
                  push-vid: 127
        state: overridden


    # Using deleted

    - name: Delete forwading domain
      ciena.saos10.saos10_fds:
        config:
          - name: remote-fd
        state: deleted
